## 🚀 启动方式

### 前端启动
```bash
cd frontend
npm install  # 首次运行需要安装依赖
npm run dev  # 启动开发服务器
```

前端将在 `http://localhost:5173` 启动

### 后端启动
```bash
cd backend
pip install -r requirements.txt  # 首次运行需要安装依赖
python main.py  # 启动后端服务器
```

后端将在 `http://localhost:8000` 启动
